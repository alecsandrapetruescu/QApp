### Usage
### Requirements
Node.js version 16.x
### Install the dependencies
```bash
npm install --omit=optional
```

#### Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
npm run dev
```

#### Lint the files

```bash
npm run lint
```

#### Build the app for production

```bash
npm run build
```
