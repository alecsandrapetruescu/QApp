## Motivation
[7 Reasons to migrate from Vuetify to Quasar Framework](https://medium.com/@charliegilmanuk/7-reasons-to-migrate-from-vuetify-to-quasar-framework-8ea47358262)

## Usage
```
// In package.json

"scripts": {
    "start": "quasar dev",


// In cli
> npm install
> npm run start
```
